# lyrics.iamtakagi.vercel.app

## Getting Started

### Environment Variables
以下の環境変数が必要です
```env
# Common
SITE_DOMAIN=

# last.fm
LASTFM_USER_ID=

# Twitter
TWITTER_ID=

# Supabase
SUPABSE_URL=
SUPABSE_ANON_KEY=
```

### Installation
自力で頑張ってください
